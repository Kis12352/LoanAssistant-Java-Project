# LoanAssistant-Java-Project
Java suvenconsultant Internship project using SWING on the financial application.
There are total four files nbproject, src/loanassistant, build.xml and manifest.mf.
There are four files in the nbproject folder named as build-impl.xml, genfiles.properties, project.properties and project.xml.
Main code is present in the LoanAssistant.java this file is present in the src/loanassistant folder.
